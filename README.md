# GestorGastos

**Aplicación Android para gestión de gastos personales**

GestorGastos es una aplicación desarrollada en **Kotlin** que permite a los usuarios registrar, organizar y controlar sus gastos personales. La app ayuda a gestionar objetivos por categoría y visualizar el estado de las finanzas de forma clara y sencilla.

## Funcionalidades

- Registro de gastos diarios
- Organización de gastos por categorías
- Visualización de objetivos por categoría
- Selección de moneda
- Interfaz sencilla y amigable

## Tecnologías utilizadas

- **Lenguaje:** Kotlin  
- **Plataforma:** Android Studio  
- **Gestión de dependencias:** Gradle  
- **Persistencia:** SQLite / Room (según implementación)  

## Rol en el proyecto

- Desarrollo principal de la lógica de la aplicación (backend de la app)  
- Implementación de validaciones y manejo de datos  
- Integración con la interfaz de usuario  
- Optimización de la gestión de gastos y objetivos
